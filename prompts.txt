In scraper/scrape.py, write a Python scraper for Costco gas prices in San Diego County.
Logic:

Maintain a data/stations.json file with discovered Costco gas stations. If it's missing or older than 30 days, rediscover by fetching the Costco warehouse locator and finding warehouses with zip codes starting "92" (San Diego County) that have gas stations on their detail pages.
For each station, fetch the warehouse page and parse regular and premium gas prices from the "Gas Station" section. The price is shown like "$5.79⁹" (the superscript 9 = 9/10 of a cent, so parse as 5.799).
Read data/prices.json, append a new record {"timestamp": ISO8601, "regular": 5.799, "premium": 6.199} to that station's history. Keep only the last 90 days of history per station to keep the file small.
Write back to data/prices.json.

Use requests + BeautifulSoup. Set a real User-Agent header. Handle each station independently so one failure doesn't kill the run. Log clearly which stations succeeded and which failed. Include a requirements.txt with pinned versions.
